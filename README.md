# Sorting_Visualiser
Sorting Visualizer is an interactive web application that demonstrates how different sorting algorithms work through real-time animations. Users can generate random arrays, adjust speed, and visualize algorithms like Bubble, Selection, Insertion, Merge, and Quick Sort, making algorithm learning easier and more engaging.

Features
Visual representation of sorting algorithms
Adjustable array size
Random array generation
Sorting speed control
Step-by-step animation
Responsive user interface
Performance comparison between algorithms
Algorithms Implemented
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
Technologies Used
HTML
CSS
JavaScript
How It Works
Generate a random array.
Select a sorting algorithm.
Adjust the speed and array size if needed.
Start the visualization.
Observe how the algorithm sorts the array through animated comparisons and swaps.
